# Portfolio Wireframe Using HTML Tables

## 📌 Project Overview

The Portfolio Wireframe is a simple and structured personal portfolio website developed using pure HTML and table layouts. The purpose of this project is to demonstrate the basic structure of a portfolio website without using advanced CSS frameworks or JavaScript.

The website presents personal information, technical skills, projects, educational qualifications, certifications, and contact details in an organized format. This project helps in understanding HTML fundamentals, table structures, hyperlinks, images, and page organization.

---

## 🎯 Project Objectives

* Create a portfolio website using only HTML.
* Practice table-based webpage layouts.
* Display personal and professional information effectively.
* Showcase skills, projects, and educational background.
* Develop a structured wireframe before creating a fully responsive portfolio.
